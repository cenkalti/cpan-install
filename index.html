You are seeing an error message like this when running a Perl script? 

Can't locate LWP/UserAgent.pm in @INC (@INC contains: /etc/perl /usr/local/lib/perl/5.12.4 /usr/local/share/perl/5.12.4 /usr/lib/perl5 /usr/share/perl5 /usr/lib/perl/5.12 /usr/share/perl/5.12 /usr/local/lib/site_perl .) at -e line 1.

BEGIN failed--compilation aborted at -e line 1.

This means you are missing the Perl module LWP::UserAgent.
(Get the module name by replacing / with :: and ignore the .pm extension)

Install a module

Install LWP::UserAgent on your system with cpanminus, a zero configuration CPAN client: 

curl -L http://cpanmin.us | perl - --sudo App::cpanminus
cpanm --sudo LWP::UserAgent

Or, if you donâ€™t have root access, you can install to ~/perl5 instead: 

$ curl -L http://cpanmin.us | perl - -l ~/perl5 local::lib App::cpanminus
$ echo 'eval $(perl -I~/perl5/lib/perl5 -Mlocal::lib)' >> ~/.bashrc
$ eval `perl -I~/perl5/lib/perl5 -Mlocal::lib`
$ cpanm LWP::UserAgent

Troubleshooting

1) cpanm says "! Can't configure the distribution. You probably need to have 'make'. See /home/maikf/.cpanm/build.log for details." Apparently, your system does not have the make command, which you need to install Perl modules. On Debian, install: sudo apt-get install build-essential
